# Askly Documentation

This folder contains technical documentation for the Askly Therapy Chatbot project.

## Contents
- `architecture.md`: High-level system architecture, data flow, and security overview
- `api.md`: API endpoints and usage (to be added)
- `features.md`: Feature descriptions and user flows (to be added)

## Project Overview
Askly is a full-stack web application for mental wellness support, built for university students. It includes:
- AI-powered chatbot (OpenAI/Gemini)
- Mood tracking
- Private journaling
- Mindfulness tools
- Secure authentication and privacy-first design

## Technology Stack
- **Frontend:** React.js, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Authentication/Data:** Firebase Auth, Firestore
- **AI:** OpenAI API (or Gemini)
